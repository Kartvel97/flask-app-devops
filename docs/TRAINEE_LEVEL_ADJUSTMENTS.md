# Упрощения для уровня Trainee

## Что выглядит слишком профессионально

### 1. Структурированное логирование (app.py)
**Текущее:** JSON логирование с полной структурой
**Для Trainee:** Простое логирование через print или basic logging

### 2. Prometheus метрики
**Текущее:** Counter, Histogram, Gauge с labels
**Для Trainee:** Простой счетчик запросов

### 3. Terraform IAM
**Текущее:** IAM роли, policies, instance profiles
**Для Trainee:** Простой EC2 без IAM (S3 backup через AWS CLI credentials)

### 4. GitHub Actions кеширование
**Текущее:** cache-from: type=gha, cache-to: type=gha
**Для Trainee:** Убрать кеширование

### 5. Мониторинг стек
**Текущее:** Prometheus + Grafana + Loki + Promtail + Alertmanager
**Для Trainee:** Только Prometheus + Grafana (без Loki)

## Рекомендации
